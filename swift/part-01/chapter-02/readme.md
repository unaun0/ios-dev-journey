# Глава 2. Фундаментальные типы данных

**Тип данных** - это множество возможных значений и операций с этими значениями.

В Swift, как и в других языках программирования, есть предустановленные фундаментальные типы данных, доступные "из коробки".

**Swift Standard Library** — это фундаментальная библиотека, которая:
- является неотъемлемой частью языка Swift — ее существование заложено в спецификацию языка;
- предоставляет базовые типы, протоколы, функции, составляющие минимальную инфраструктуру для написания кода на Swift;
- автоматически подключается компилятором к каждой программе — не требует явного импорта;
- поставляется в составе Swift toolchain (набора инструментов компилятора).

## 2.1 Предназначение типов данных

**Неявное определение типа** - Swift автоматически определяет тип параметра на основе инициализируемого значения.

```
let someText = "Я учу Свифт"  // тип String определен неявно
```

**Явное определение типа** — разработчик самостоятельно указывает тип данных после имени через двоеточие.

```
var elfsName: String = "Эанор"  // тип String указан явно
```

| Правило                 | Описание                                                       |
|-------------------------|----------------------------------------------------------------|
| **Обязательность типа** | У любого параметра должен быть определен тип (явно или неявно) |
| **Строгая типизация**   | Тип параметра нельзя изменить после объявления                 |
| **Инициализация**       | Параметр должен иметь значение к моменту первого использования |

```
var str: String      // объявление без значения — допустимо
str = "Hello"        // инициализация перед использованием — обязательно
print(str)           // использование — OK
```

## 2.2 Числовые типы данных

### Целочисленные типы данных

**Целые числа** — это числа, у которых отсутствует дробная часть, например, 81 или 18. 

В Swift целочисленные типы делятся на две группы: *знаковые (signed)* и *беззнаковые (unsigned)*.

| Вид | Описание | Примеры |
|-----|----------|---------|
| **Знаковые (signed)** | Позволяют хранить ноль, положительные и отрицательные значения | `Int`, `Int8`, `Int16`, `Int32`, `Int64` |
| **Беззнаковые (unsigned)** | Позволяют хранить только ноль и положительные значения | `UInt`, `UInt8`, `UInt16`, `UInt32`, `UInt64` |

**Числовой литерал** — это фиксированная последовательность цифр, начинающаяся либо с цифры, либо с префикса "+" или "-".

```
2      // положительный литерал
-64    // отрицательный литерал
+18    // положительный литерал с явным знаком
```

По умолчанию целочисленные литералы в Swift имеют тип Int, если не указано иное.

| Тип | Диапазон |
|-----|----------|
| **Знаковый (signed)** | от –2ⁿ⁻¹ до 2ⁿ⁻¹ – 1 |
| **Беззнаковый (unsigned)** | от 0 до 2ⁿ – 1 |

где n — разрядность (8, 16, 32, 64).

| Тип | Разрядность | Минимальное значение | Максимальное значение |
|-----|-------------|----------------------|----------------------|
| `Int` | 64 бита* | -9223372036854775808 | 9223372036854775807 |
| `Int8` | 8 бит | -128 | 127 |
| `Int16` | 16 бит | -32768 | 32767 |
| `Int32` | 32 бита | -2147483648 | 2147483647 |
| `Int64` | 64 бита | -9223372036854775808 | 9223372036854775807 |
| `UInt` | 64 бита* | 0 | 18446744073709551615 |
| `UInt8` | 8 бит | 0 | 255 |
| `UInt16` | 16 бит | 0 | 65535 |
| `UInt32` | 32 бита | 0 | 4294967295 |
| `UInt64` | 64 бита | 0 | 18446744073709551615 |

Тип `Int` является основным для работы с целыми числами и зависит от разрядности платформы: на современных устройствах (64-битных системах) это, как правило, 64-битный тип (`Int64`).

Для удобства чтения больших чисел можно использовать символ `_` в качестве разделителя:

```
let million = 1_000_000
```

Swift также поддерживает разные системы счисления:

```
let decimal = 10      // десятичная
let binary = 0b1010   // двоичная
let octal = 0o12      // восьмеричная
let hex = 0xA         // шестнадцатеричная
```
Важной особенностью языка является строгая типизация: **все операции должны выполняться между значениями одного типа**. Swift не выполняет неявное преобразование чисел, поэтому при необходимости требуется явное приведение типов:

```
let a: Int8 = 10
let b: Int16 = 20
let c = Int16(a) + b  // приведение типов
```

### Объектные возможности Swift

Swift обладает важной особенностью: в этом языке практически всё является объектами или ведёт себя как объекты.

**Объект** — это сущность, объединяющая данные и поведение.

Объект характеризуется:
- *свойствами* — данными, описывающими его состояние;
- *методами* — функциями, определяющими поведение объекта и операции над его данными.

Каждое свойство и метод имеют имя, через которое к ним осуществляется доступ. В Swift доступ к свойствам и методам выполняется с помощью точечной нотации:

```
Объект.свойство
Объект.метод()
```

Например:

```
ProductMachine.maxCapacity
ProductMachine.dispense("chocolate")
```

**Метод** — это функция, которая связана с конкретным объектом и выполняется в его контексте.

Поскольку в Swift всё рассматривается как объект, даже значения базовых типов данных (например, чисел) обладают свойствами и методами. Так, целочисленные типы данных представляют собой объекты, описывающие целые числа с определёнными диапазонами значений.

Каждый числовой тип имеет встроенные свойства, позволяющие получить минимальное и максимальное допустимые значения. Например:

```
let minInt8 = Int8.min   // -128
let maxInt8 = Int8.max   // 127
```

```
let minUInt8 = UInt8.min // 0
let maxUInt8 = UInt8.max // 255
```

В Swift предусмотрена защита от переполнения. **Если результат операции выходит за допустимый диапазон типа, возникает ошибка времени выполнения (runtime error)**:

```
let max = Int8.max  // 127
// let overflow = max + 1  // ошибка переполнения
```

### Числа с плавающей точкой

Помимо целых чисел, при разработке приложений используются числа. Например, `3.14` и `-192.884022`.

**Дробные числа** называются *числами с плавающей точкой*.

Для хранения дробных чисел в большинстве случаев используются всего два типа данных: `Float` и `Double`, оба являются знаковыми (позволяют хранить положительные и отрицательные значения).

- `Float` — позволяет хранить 32-битное число с плавающей точкой, содержащее до 6 знаков в дробной части.
- `Double` — позволяет хранить 64-битное число с плавающей точкой, содержащее до 15 знаков в дробной части.

При передаче дробного значения без явного указания типа Swift всегда самостоятельно определяет тип данных `Double`.

```
// дробное число типа Float с явным указанием типа
let numFloat: Float = 104.3

// дробное число типа Double с явным указанием типа
let numDouble: Double = 8.36

// дробное число типа Double с неявным указанием типа
let someNumber = 8.36
```

Форматы записи:
- **Десятичные** (по умолчанию)
- **Шестнадцатеричные** (с префиксом 0x)

**Важно**: по обе стороны от точки используется одна и та же система счисления

**Экспонента**:
- Для десятичных чисел — e (степень 10)
- Для шестнадцатеричных — p (степень 2)

```
let decimal = 12.1875
let exponent = 1.21875e1 // 12.1875
let hex = 0xC.3p0 // 12.1875
```

**Смысл экспоненты**:
- 1.2e3 = 1.2 × 10³
- 0xFp2 = 15 × 2²

### Арифметические операторы в Swift

Общие положения:
- Арифметические операции выполняются с помощью операторов.
- Операции применимы к числовым типам (Int, Float, Double).
- Тип результата совпадает с типом операндов.
- Операции можно выполнять только между значениями одного типа.

| Оператор | Вид                  | Описание                                     | Пример                                              |
| -------- | -------------------- | -------------------------------------------- | --------------------------------------------------- |
| `+`      | Бинарный             | Складывает два операнда (`a + b`)            | `let res = 1 + 2 // 3`                              |
| `+`      | Унарный (префиксный) | Возвращает значение без изменений            | `let res = +3`                                      |
| `-`      | Бинарный             | Вычитает второй операнд из первого (`a - b`) | `let res = 4 - 3 // 1`                              |
| `-`      | Унарный (префиксный) | Меняет знак числа (`-a`)                     | `let neg = -5`                                      |
| `*`      | Бинарный             | Умножает два операнда                        | `let res = 4 * 5 // 20`                             |
| `/`      | Бинарный             | Деление операндов                            | `let res = 20 / 4 // 5`<br>`let res2 = 19 / 4 // 4` |
| `%`      | Бинарный             | Остаток от деления (только Int)              | `let res = 19 % 4 // 3`                             |

Особенности:
- Swift не имеет операторов `++` и `--`
- При смешении типов (`Int` + `Double`) возникает ошибка → требуется явное приведение типов
- Деление целых чисел: не округляется, дробная часть просто отбрасывается

Работа с Float / Double: 
- Поддерживают те же операторы: `+`, `-`, `*`, `/`
- Результат = дробное число

```
let a = 3.14
let b = 1.1
let res = a + b
```

Остаток для дробных чисел - `truncatingRemainder(dividingBy:)`:

```
let result = 3.14.truncatingRemainder(dividingBy: 1.1)
```

Проверка кратности - `isMultiple(of:)`:

```
let isEven = 4.isMultiple(of: 2) // true
```

### Приведение числовых типов данных

Общее правило:
- Арифметические операции выполняются только между значениями одного типа
- При работе с разными типами требуется явное приведение

В Swift используется приведение типов через инициализаторы типов:

- `Int(_:)` — преобразование к `Int`
- `Double(_:)` — преобразование к `Double`
- `Float(_:)` — преобразование к `Float`

```
let a = Int(2)
let b = Double(2)
let c = Float(2)
```

При операциях между разными типами **необходимо привести один тип к другому**:

```
let numberInt = 19
let numberDouble = 3.13

let resD = Double(numberInt) * numberDouble
let resI = numberInt * Int(numberDouble)
```

Особенности преобразования:
- При преобразовании Int → Double данные не теряются
- При преобразовании Double → Int дробная часть отбрасывается (без округления)
- Результат вычислений зависит от выбранного типа приведения

Пример потери точности:

```
let value = Int(3.99) // 3
```

### Составной оператор присваивания

- Комбинация: арифметический оператор и `=`
- Используется для сокращения записи операций над переменной

```
x += y
x -= y
x *= y
x /= y
x %= y
```

Пример:
```
var count = 19

count += 5   // count = count + 5 → 24
count *= 3   // 72
count -= 3   // 69
count %= 8   // 5
```

### Тип данных Decimal и точность операций

Проблема точности чисел с плавающей точкой:
- `Float` и `Double` хранят числа в двоичном виде
- Некоторые десятичные числа (например, 0.1) не могут быть точно представлены в двоичной системе
- Возникает погрешность округления

```
var myWallet: Double = 0
let income: Double = 0.1

myWallet += income
myWallet += income
myWallet += income

print(myWallet) // 0.30000000000000004
```

Причина:
- Число 0.1 в двоичной системе — бесконечная дробь
- Компьютер хранит приближение
- Ошибка накапливается при операциях

Когда это критично:
- Финансовые расчёты
- Любые вычисления, где важна точная десятичная сумма

Решение: **Использовать тип `Decimal` (из `Foundation`):**

```
import Foundation

var wallet: Decimal = 0
let income: Decimal = 0.1

wallet += income
wallet += income
wallet += income

print(wallet) // 0.3
```

Особенности `Decimal`:
- Работает с десятичной системой (база 10)
- Обеспечивает высокую точность
- Медленнее Float и Double
- Требует `import Foundation`

`Foundation` — это базовая системная библиотека (фреймворк) в экосистеме Swift / Apple Platform SDK, которая предоставляет готовые инструменты для работы с данными и типами, не входящими в стандартный набор языка.

Foundation включает:
- дополнительные типы данных (`Decimal`, `Date`, `Data`)
- работу со строками и их обработку
- коллекции и утилиты
- форматирование данных
- работу с файлами и JSON
- базовые алгоритмы и вспомогательные функции

## 2.3 Строковые типы данных

**Строка** — это последовательность символов.

Минимальная единица строки — **символ**.

Компьютер хранит символы в виде чисел (0 и 1) через стандарты кодирования, например *Unicode*.

**Unicode** - Международный стандарт кодирования символов:
- Каждому символу соответствует кодовая точка (code point).
- Пример: 'a' → 97 (десятичная), 0x61 (шестнадцатеричная)
- Обеспечивает корректную передачу текста между устройствами.
- Поддерживает все языки, знаки, эмодзи.
- Для преобразования кодовой точки в двоичный вид используются кодировки: UTF-8, UTF-16, UTF-32.

### Основные типы строк в Swift

**`Character`**  — один символ: `let char: Character = "a"`

**`String`** — последовательность символов любой длины:
```
let stringOne: String = "Dragon"
let language = "Swift" // тип String определяется неявно
```

**Пустая строка** — строковый литерал без символов: `""`
- Можно создать через инициализатор: `String()`
- Пустая строка не может быть `Character`.

**Многострочные строки** обрамляются тремя двойными кавычками `"""`.

```
let longString = """
Это многострочный
строковый литерал
"""
```
Открывающие и закрывающие кавычки должны быть на отдельной строке.

### Приведение к строковому типу

Функция `String(_)` превращает значения других типов в строку:
```
let someDoubleNumber = 74.22
let someStringNumber = String(someDoubleNumber) // "74.22"
```

### Объединение строк

1. *Интерполяция* — вставка переменных/выражений в строку:
```
let name = "Дракон"
let hello = "Привет, меня зовут \(name)!"
```

2.*Конкатенация* — объединение через `+`:
```
let firstText = "Мой вес "
let weight = 12.4
let secondText = " тонны"
let resultText = firstText + String(weight) + secondText
```

### Сравнение строк

- Используется оператор `==`:

```
let firstString = "Swift"
let secondString = "Objective-C"
let anotherString = "Swift"
```

```
firstString == secondString // false
firstString == anotherString // true
```

### Юникод-скаляры

Специальная запись символа через его кодовую точку:

```
let myCharOverUnicode: Character = "\u{041A}" // "К"
let stringOverUnicode = "\u{41C}\u{438}\u{440}\u{20}\u{412}\u{430}\u{43C}\u{21}" // "Мир Вам!"
```

## 2.4 Логический тип данных

Тип данных ('Bool') для хранения логических значений:
- `true` — истина;
- `false` — ложь.

```
let isDragon = true
let isKnight: Bool = false
```

Swift не позволяет подменять тип `Bool` другими типами. Например, нельзя присвоить строку или число вместо `Bool`.

`Bool` чаще всего используется в условных конструкциях:
- Если true → выполняется блок `if`
- Если false → выполняется блок `else`

```
var isDragon = true

if isDragon {
    print("Привет, Дракон!")
} else {
    print("Привет, Тролль!")
}
```

### Логические операторы

- **Логическое НЕ (`!`)** - инвертирует значение:
```
let someBool = true
!someBool // false
```

- **Логическое И (`&&`)** - возвращает `true`, если оба условия `true`:

```
true && true   // true
true && false  // false
```

- **Логическое ИЛИ (`||`)** - возвращает `true`, если хотя бы одно условие `true`:
```
true || false  // true
false || false // false
```

Логические операторы можно комбинировать: `let result = true && false || false // false`

Порядок выполнения можно задавать скобками: `let result = true && (false || true) // true`

### Операторы сравнения

Операторы сравнения возвращают `true` или `false`:

- `==` — равно
- `!=` — не равно
- `>` — больше
- `<` — меньше
- `>=` — больше или равно
- `<=` — меньше или равно

```
1 > 2       // false
2 != 2      // false
(1 + 1) < 3 // true
5 >= 1      // true
```

## 2.5 Псевдонимы типов

**Псевдоним типа** (`typealias`) — это дополнительное имя для существующего типа данных.
- Используется для упрощения и повышения читаемости кода.
- Не создает новый тип, а лишь даёт новое имя уже существующему.

```
typealias AgeType = UInt8
var myAge: AgeType = 29
```

Один тип может иметь несколько псевдонимов:
```
typealias TextType = String
typealias WordType = String
typealias CharType = String

let someText: TextType = "Это текст"
let someWord: WordType = "Слово"
let someChar: CharType = "Б"
```

Псевдоним полностью повторяет возможности исходного типа.
Можно обращаться к свойствам и методам типа через псевдоним:
```
typealias AgeType = UInt8
let maxAge = AgeType.max // 255
```

**Псевдоним** — это ссылка на тип, а не новый тип.

Можно использовать с любыми типами:
- простыми (Int, String, Bool)
- составными (кортежи и др.)

Не влияет на поведение данных, только на имя.

## 2.6. Дополнительные сведения о типах данных

### Как узнать тип данных?

- Через Xcode: навести курсор на переменную с зажатой клавишей Option

- С помощью функции `type(of:)`

```
let myVar = 3.54
print(type(of: myVar)) // Double
```

### Категории типов данных

Типы данных могут принадлежать к определённым категориям (протоколам), которые определяют их возможности.

#### Comparable (сопоставимые типы)

Тип называется `Comparable`, если его значения можно сравнивать с помощью: `<`, `<=`, `>`, `>=`.

Примеры:
`Int`, `Double`, `String` — сопоставимые
`Bool` — не сопоставимый

```
"string1" < "string2" // true
true < false          // ошибка
```

**!!! Важно !!!:**
- Операторы `==` и `!=` относятся к другой категории — `Equatable`.
- `Comparable ≠ Equatable` (это разные требования)

#### Hashable (хешируемые типы)

Тип называется `Hashable`, если для его значения можно вычислить хеш.

`Хеш` - уникальное числовое значение, получаемое из данных.

Представь:
- у тебя есть строка "Строка текста"
- система превращает её в число (например: 109231433150392402)
- Это число и есть хеш — как “ID” для значения.

**Хеш используется для быстроты.**

Например:
- поиск в коллекциях (Set, Dictionary);
- сравнение объектов;
- ускорение операций.

Вместо того чтобы сравнивать строки посимвольно:

```
"Строка текста" == "Строка текста"
```

Swift может:

```
посчитать хеш
сравнить числа
(это быстрее)
```



**!!! Важно !!!:**
- Все фундаментальные типы Swift являются `Hashable`,
- Хеш зависит от внутренних алгоритмов и может меняться.
